# AquaCase Expert 🐟

**Project Akhir Sistem Pakar**

AquaCase Expert adalah sebuah Sistem Pakar yang dibangun untuk **Mendeteksi Penyakit Pada Ikan Air Tawar**. Sistem ini dirancang menggunakan gabungan beberapa metode kecerdasan buatan untuk memberikan hasil diagnosa dan rekomendasi penanganan yang akurat.

## 🔬 Metode yang Digunakan
Sistem pakar ini menggunakan integrasi dari beberapa metode komputasi:
- **Rule-Based Reasoning (RBR)** dengan **Certainty Factor (CF)**
- **Case-Based Reasoning (CBR)** dengan algoritma **Nearest Neighbor**
- **Knowledge Base (KB)** untuk penyimpanan aturan dan kasus pakar
- **Fussion** (Penggabungan konklusi/hasil penalaran dari RBR dan CBR)
