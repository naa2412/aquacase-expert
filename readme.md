# AquaCase Expert

Folder ini berisi data knowledge base untuk sistem pakar diagnosis penyakit ikan air tawar berbasis Hybrid Case-Based Reasoning (CBR) dan Rule-Based Reasoning (RBR).

## Sumber Data

Knowledge base diadaptasi dari tesis Hisma Abduh tentang diagnosis penyakit ikan air tawar menggunakan Rule-Based Reasoning dan Case-Based Reasoning. Data penyakit ikan berasal dari Balai Pengembangan Teknologi Kelautan dan Perikanan Argomulyo, Yogyakarta.

## Cakupan Knowledge Base

Data mencakup:
* 9 jenis ikan
* 45 gejala
* 13 penyakit
* 142 data kasus CBR
* 30 data uji
* 458 rule RBR
* solusi pengobatan dan penanganan dari masing-masing penyakit
